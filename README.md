# EntrySorter

EntrySorter is a simple Visual Basic Windows Forms application for managing and sorting entries.  
It allows users to add, search, delete, and sort items in a list with an easy‑to‑use interface.

## Features
- Add new entries to the list
- Search and filter existing entries
- Delete entries by selection or search
- Sort entries alphabetically
- Clean, minimal Windows Forms UI
